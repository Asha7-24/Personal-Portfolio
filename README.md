# 🌐 Personal Portfolio Website

A modern and responsive personal portfolio website built using **HTML, CSS, and JavaScript** to showcase my skills, projects, and professional profile.

## 🚀 Live Features

✨ Modern Dark UI Design  
✨ Aurora Gradient Background Effects  
✨ Responsive Layout for Mobile & Desktop  
✨ Typing Text Animation  
✨ Smooth Scrolling Navigation  
✨ Active Navigation Highlighting  
✨ Scroll Reveal Animations  
✨ Interactive Hover Effects  
✨ Back-to-Top Button  
✨ Glassmorphism Cards Design
